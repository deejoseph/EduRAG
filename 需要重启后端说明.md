# 需要重启后端服务

## 当前问题

您遇到的以下问题都是因为**后端服务没有重启**导致的：

### 1. 模型显示问题
- **现象**: 首页显示 `qwen3:8b`，但config.yaml配置的是 `gemma3:4b`
- **原因**: 后端启动时读取了旧配置，仍在使用 `qwen3:8b`
- **解决**: 重启后端后会正确读取config.yaml中的配置

### 2. 系统设置中无法切换模型
- **现象**: 点击"切换模型"下拉框，没有显示任何可选模型
- **原因**: `/system/model` API返回404（新代码未加载）
- **验证**: 
  ```bash
  curl http://localhost:5000/system/model
  # 返回: 404 Not Found
  ```
- **解决**: 重启后API会生效，并显示Ollama实际安装的5个模型

### 3. 命题热点搜索结果重复
- **现象**: 搜索2次，每次都返回相同的21个题目
- **原因**: 
  1. 使用了缓存数据（相同缓存 = 相同结果）
  2. LLM提示词优化代码未加载
- **解决**: 重启后，新的提示词会增加命题多样性

### 4. "一键收藏"功能位置说明
- **位置**: 在"引导练习"页面（原写作训练），不是命题热点页面
- **路径**: 侧边栏 → 引导练习 → 输入题目 → 点击"一键收藏到题库"按钮
- **功能**: 将自定义题目快速添加到题库

## 重启步骤

```bash
# 1. 停止当前运行的后端服务（按 Ctrl+C）

# 2. 重新启动后端
```
